📱 <b>iPhone / iPad</b>

У нас два варианта VPN — у каждого <u>своё приложение</u>:

<b>① Основной VPN (AmneziaWG)</b>
Приложение: <b>AmneziaWG</b> (или AmneziaVPN) — из App Store.
• Проще всего: открой <b>Личный кабинет</b> → «Основной VPN» → iPhone/iPad → <b>отсканируй QR</b> прямо в приложении AmneziaWG.
• Или: скачай файл <code>.conf</code> → приложение «Файлы» → нажми на файл → «Поделиться» → AmneziaWG.

<b>② VPN при блокировках (VLESS)</b> — если основной не идёт по LTE/5G.
Приложение: <b>Streisand</b>, <b>Happ</b> или <b>V2Box</b> — из App Store.
• Личный кабинет → «VPN при блокировках» → выбери оператора → <b>отсканируй QR</b> в приложении (или скопируй ссылку <code>vless://</code> и вставь: «+» → «Импорт из буфера»).

⚠️ <b>Куда что вставлять:</b>
• ссылка <code>vless://</code> → только Streisand / Happ / V2Box (НЕ в AmneziaWG)
• конфиг или QR AmneziaWG → только в AmneziaWG (НЕ в V2Box)
