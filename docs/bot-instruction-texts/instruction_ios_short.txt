📱 <b>iPhone / iPad</b>

В боте: <b>«📲 Получить VPN»</b> — три способа подключения.

<b>🔗 Быстрый VPN (рекомендуется)</b>
1. Поставь <b>Streisand</b>, <b>Happ</b> или <b>V2Box</b> (App Store).
2. В боте: «Получить VPN» → «Быстрый VPN» → отсканируй QR в приложении (или скопируй ссылку → «+» → «Импорт из буфера»).
3. Приложение само выберет рабочий сервер.

<b>📲 Резервный VPN (AmneziaWG)</b>
1. Поставь <b>AmneziaWG</b> из App Store.
2. В боте: «Получить VPN» → «Резервный VPN» → iOS → получишь файл <code>.conf</code>.
3. Нажми на файл → «Поделиться» → <b>AmneziaWG</b> → «Создать из файла».
4. Включи туннель.

<b>📡 Мобильный VPN</b>
Если оператор режет интернет — «Получить VPN» → «Мобильный VPN» → выбери оператора. Получишь <code>vless://</code> для Streisand / Happ / V2Box.

⚠️ Не путать форматы: <code>vless://</code> → только Streisand/Happ/V2Box; файл <code>.conf</code> → только AmneziaWG.
