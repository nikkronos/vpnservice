💻 <b>ПК (Windows / macOS / Linux)</b>

В боте: <b>«📲 Получить VPN»</b> — три способа подключения.

<b>🔗 Быстрый VPN (рекомендуется)</b>
1. Поставь <b>Happ</b>: <a href="https://happ.su/">happ.su</a>.
2. В боте: «Получить VPN» → «Быстрый VPN» → скопируй ссылку.
3. В Happ: «+» → «Импорт из буфера» → вставь ссылку.
4. Приложение само выберет рабочий сервер.

<b>📲 Резервный VPN (AmneziaWG)</b>
1. Поставь <b>AmneziaVPN</b>: <a href="https://amnezia.org/">amnezia.org</a>.
2. В боте: «Получить VPN» → «Резервный VPN» → ПК → получишь файл <code>.conf</code>.
3. В AmneziaVPN: «+» → «Импорт из файла» → выбери конфиг.
4. Включи туннель.
