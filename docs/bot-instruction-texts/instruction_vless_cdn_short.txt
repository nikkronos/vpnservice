📶 <b>Особый канал для Yota / Мегафон</b>

Этот профиль маскируется под обычное HTTPS-соединение к cloud.mail.ru и работает на твоём операторе <u>даже при белых списках РКН</u> — когда обычный VPN отваливается.

<b>Шаги:</b>
1. Бот пришлёт ссылку <code>vless://...</code> вторым сообщением — скопируй её целиком.
2. Установи любое приложение с поддержкой VLESS:
   • iOS: <b>Streisand</b>, <b>Happ</b> или <b>V2Box</b> (App Store)
   • Android: <b>Hiddify</b> или <b>v2rayNG</b> (Google Play)
3. В приложении: «+» → «Импорт из буфера» → вставь ссылку → подключись.

<b>Важно:</b> вставлять только в Streisand/Happ/V2Box/Hiddify. В AmneziaWG эта ссылка не работает — там другой формат.

🌐 Личный кабинет (одна ссылка-подписка для всех устройств): <code>https://supportkronos.online:8443/recovery</code>
