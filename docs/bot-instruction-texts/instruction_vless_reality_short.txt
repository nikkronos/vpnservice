📶 <b>Мобильный интернет (LTE/5G)</b>

Если AmneziaWG работает по Wi‑Fi, но не по сотовой — используй этот резервный профиль (TCP, маскировка под HTTPS).

<b>Шаги:</b>
1. Бот пришлёт ссылку <code>vless://...</code> вторым сообщением — скопируй её целиком.
2. Установи любое приложение с поддержкой VLESS:
   • Android: <b>Hiddify</b> или <b>v2rayNG</b>
   • iOS: <b>Hiddify</b>, <b>FoXray</b> или <b>V2Box</b> (ищи в App Store, список может меняться)
3. В приложении: «+» → «Импорт из буфера» → вставь ссылку → подключись.

<b>Важно:</b> ссылка <code>vless://</code> — универсальный формат, работает в любом совместимом клиенте. Если одного нет в сторе — поставь другой.
Этот профиль только для мобильной сети, на ПК ничего менять не нужно.
