💻 <b>Windows / macOS / Linux</b>

Два варианта VPN — у каждого <u>своё приложение</u>:

<b>① Основной VPN (AmneziaWG)</b>
1. Скачай <b>AmneziaVPN</b>: <a href="https://amnezia.org/en/downloads">amnezia.org/en/downloads</a>
2. В боте: «Получить VPN» → ПК — придёт файл <code>.conf</code>.
3. В AmneziaVPN: <b>+</b> → <b>Импорт из файла</b> → выбери конфиг (на ПК можно и «Добавить по ссылке»).
4. Включи туннель — VPN работает.

<b>② Мобильный резерв (VLESS)</b> — если основной не идёт.
Приложение: <b>Hiddify</b> (<a href="https://hiddify.com">hiddify.com</a>) или v2rayNG.
Личный кабинет → «Мобильный резерв» → скопируй ссылку <code>vless://</code> → в Hiddify: «+» → «Добавить из буфера».

⚠️ <b>Куда что вставлять:</b> конфиг AmneziaWG → в AmneziaVPN; ссылка <code>vless://</code> → в Hiddify/v2rayNG. Не путать — это разные ссылки.

⚠️ <b>AmneziaVPN: Error 103 «Фоновая служба не запущена»:</b>
1. Закрой AmneziaVPN полностью (Диспетчер задач → заверши все процессы Amnezia*).
2. Запусти AmneziaVPN <b>«Запуск от имени администратора»</b> (правой кнопкой по ярлыку).
3. Не помогло — перезагрузи Windows и снова запусти от админа.
4. Всё ещё нет — переустанови AmneziaVPN. Либо временно перейди на «Мобильный резерв» (VLESS) в Hiddify — это <u>отдельная</u> ссылка из Личного кабинета, не тот же конфиг.
