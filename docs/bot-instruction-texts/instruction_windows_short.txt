💻 <b>Windows</b>

<b>Шаги подключения:</b>
1. Скачай <b>AmneziaVPN</b>: <a href="https://amnezia.org/en/downloads">amnezia.org/en/downloads</a>
2. В боте нажми <b>Получить конфиг</b> — бот пришлёт файл <code>.conf</code>.
3. В AmneziaVPN: нажми <b>+</b> → <b>Импорт из файла</b> → выбери скачанный файл.
4. Включи туннель — VPN работает.

⚠️ <b>Если AmneziaVPN выдаёт Error 103 «Фоновая служба не запущена»:</b>
1. Закрой AmneziaVPN полностью (через Диспетчер задач, заверши все процессы Amnezia*).
2. Запусти AmneziaVPN через <b>«Запуск от имени администратора»</b> (правой кнопкой по ярлыку).
3. Если не помогло — перезагрузи Windows и снова запусти от админа.
4. Если всё ещё не работает — переустанови AmneziaVPN или используй <a href="https://hiddify.com">Hiddify</a> (тот же конфиг подойдёт).
