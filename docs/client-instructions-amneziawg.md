# Инструкция для пользователей: подключение через AmneziaWG (eu1)

VPN работает из России через протокол AmneziaWG на сервере 185.21.8.91 (eu1). Один клиент **AmneziaVPN** на ПК и на телефоне/планшете.

---

## На ПК (Windows / macOS)

1. Скачать приложение: [amnezia.org — Загрузки](https://amnezia.org/en/downloads). Версия не ниже 4.8.12.7.
2. Установить и открыть AmneziaVPN.
3. **Вариант А — у тебя уже есть доступ к серверу по SSH:** нажать «Добавить свой сервер», ввести адрес **185.21.8.91**, логин и ключ (или пароль) SSH. В приложении нажать «Установить» у протокола AmneziaWG (если ещё не установлен), создать подключение и подключаться.
4. **Вариант Б — тебе прислали конфиг/ссылку:** в AmneziaVPN нажать «Импортировать» или «Добавить по ссылке», вставить конфиг или ссылку. Подключиться выбранным профилем.
5. Галочку «Включить обфускацию WireGuard» для конфига AmneziaWG можно не трогать (обфускация уже в протоколе) или включить, если провайдер режет.

## На iPhone / iPad

### Какие приложения официальные

- **AmneziaWG** (разработчик Privacy Technologies) — отдельное приложение только для протокола AmneziaWG. Официальное от проекта Amnezia, в части регионов может быть доступно, когда полное приложение скрыто.
- **AmneziaVPN** или **DefaultVPN** (в описании может быть AMNEZIA) — полное приложение с поддержкой нескольких протоколов. Если в App Store видишь только AmneziaWG — это нормально: им и можно пользоваться.

Иконка у AmneziaWG другая (отдельный продукт) — это не подделка.

### QR-код не подходит («Недопустимый QR-код»)

С десктопа AmneziaVPN часто экспортирует конфиг в формате, который приложение **AmneziaWG** на телефоне не принимает как «конфигурацию AmneziaWG», поэтому сканирование выдаёт ошибку. Надёжный способ — перенос **файлом**.

### Надёжный способ: конфиг файлом

1. **На ПК** в AmneziaVPN: нажать «Поделиться» (Share) у нужного подключения AmneziaWG → выбрать формат экспорта **.conf** (для WireGuard/AmneziaWG) → сохранить файл или «Поделиться настройками». Отправить себе этот **.conf** файл (Telegram, почта, облако).
2. **На iPhone/iPad** скачать/сохранить .conf файл (например в «Файлы» или «Загрузки»).
3. **Импорт через «Поделиться», а не через «Импорт» в приложении:** открыть приложение **«Файлы»** → найти нужный **.conf** файл (для AmneziaWG подходит именно .conf, не .vpn) → **долгое нажатие** на файл → **«Поделиться»** → в списке приложений выбрать **AmneziaWG**. Файл откроется в AmneziaWG, туннель добавится. Останется включить подключение.

Если из AmneziaWG нажать «Создать из файла» и откроется «Файлы», но файл не выбирается — не выбирай файл там, а закрой пикер и сделай импорт через «Файлы» → долгое нажатие на .conf → «Поделиться» → AmneziaWG.

Так обходим и длинный ключ (копировать не нужно), и несовместимый QR, и проблему с выбором файла в пикере.

### Если есть полное приложение (AmneziaVPN / DefaultVPN)

В нём может быть пункт «Добавить сервер» → «У меня есть данные для подключения» → «Файл конфигурации, ключ или QR-код». Тогда тот же .conf файл можно открыть там. QR с десктопа иногда распознаётся только полным приложением, не AmneziaWG.

## Если перестало работать

- Проверить интернет и переподключиться.
- Если РКН начал блокировать AmneziaWG — владелец может добавить на сервер протокол Xray (в том же приложении Amnezia) и выдать новый конфиг; приложение менять не нужно.
