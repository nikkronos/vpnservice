# Инструкция: как подключиться к VPN на iPhone/iPad

## Что понадобится

- Файл конфигурации WireGuard (`.conf`) или его текст, выданный администратором или через Telegram‑бота

## Шаг 1: Установка WireGuard

1. Установи приложение **WireGuard** из App Store на iPhone или iPad.
2. Открой приложение.

## Шаг 2: Импорт конфига

**Вариант А — из файла**

1. Сохрани присланный файл `.conf` на устройство (например, через «Файлы» или из сообщения в Telegram).
2. В WireGuard нажми **«+»** → **«Create from file or archive»** (Создать из файла или архива).
3. Выбери сохранённый файл `.conf`.

**Вариант Б — из буфера обмена**

1. Скопируй весь текст конфига (от `[Interface]` до конца).
2. В WireGuard нажми **«+»** → **«Create from tunnel archive»** или вставь из буфера, если приложение предлагает такой вариант.

**Вариант В — по QR-коду (если администратор прислал QR)**

1. В WireGuard нажми **«+»** → **«Scan QR code»**.
2. Отсканируй QR-код с конфигом.

## Шаг 3: Подключение

1. В списке туннелей выбери нужный (например, `iphone` или `friend-iphone`).
2. Включи переключатель — статус сменится на «Active», появится значок VPN в статус-баре.

## Шаг 4: Какой профиль для чего

- **Обычный VPN** (профиль от бота, например `client1`) — весь трафик через сервер, без обязательного обхода блокировок ChatGPT.
- **VPN + GPT** (наш профиль, например `iphone`, `friend-iphone`) — тот же сервер, но HTTP/HTTPS идёт через Shadowsocks: подходит для ChatGPT и обхода блокировок; YouTube и другие сервисы могут вести себя иначе в зависимости от блокировок.

На устройстве можно иметь оба профиля и переключаться между ними по необходимости.

## Отключение

В приложении WireGuard выключи переключатель у активного туннеля.

## Telegram отдельно (без VPN)

Если нужен только Telegram при блокировках — используй MTProto‑прокси по отдельной ссылке (Настройки → Данные и хранилище → Прокси‑серверы). Инструкция: `docs/mtproto-client-instructions.md`.

## Проблемы

- **Ошибка разрешения DNS**: в конфиге должен быть указан реальный IP сервера в `Endpoint`, а не заглушка типа `SERVER_PUBLIC_IP`.
- **Не подключается**: проверь интернет и доступность сервера (порт 51820/UDP).
- **ChatGPT не открывается**: включи профиль «VPN + GPT», а не только обычный VPN.
