# Инструкция: как подключиться к VPN на ПК (Windows)

## Что понадобится

- Файл конфигурации WireGuard (`.conf`), выданный администратором или через Telegram‑бота
- При необходимости обхода блокировок (ChatGPT) — отдельный профиль или инструкция по Shadowsocks

## Шаг 1: Установка WireGuard

1. Скачай клиент WireGuard для Windows: https://www.wireguard.com/install/
2. Установи программу и запусти её.

## Шаг 2: Импорт конфига

1. В окне WireGuard нажми **«Добавить туннель»** → **«Импортировать из файла»** (или **Import tunnel(s) from file**).
2. Выбери файл `.conf`, который тебе прислали.
3. Туннель появится в списке (например, `client1` или `friend-pc`).

## Шаг 3: Подключение

1. Выбери нужный туннель в списке.
2. Нажми **«Подключить»** (или включи переключатель).
3. В статусе должно появиться «Подключен»; в интерфейсе будет показан виртуальный IP (например, `10.1.0.x`).

## Шаг 4 (опционально): Shadowsocks для ChatGPT

Если тебе выдали отдельный профиль «VPN + GPT» и на ПК нужен обход блокировок ChatGPT:

1. Убедись, что подключён нужный WireGuard‑профиль (тот, что для GPT).
2. Либо используй только этот профиль — на сервере твой трафик уже идёт через Shadowsocks для HTTP/HTTPS.
3. Либо, если администратор выдал настройки отдельного клиента Shadowsocks для Windows — установи клиент (например, Shadowsocks-Windows), введи сервер, порт, пароль и метод шифрования по инструкции.

## Нотация профилей (для справки)

- **VPN RU** / **Обычный VPN** — профиль от бота или `client1`: быстрый VPN, трафик идёт напрямую через сервер.
- **VPN GPT** — профиль с обходом блокировок: тот же сервер, но HTTP/HTTPS идёт через Shadowsocks (ChatGPT и др. работают).

## Отключение

В клиенте WireGuard нажми **«Отключить»** для выбранного туннеля.

## Проблемы

- **Не подключается**: проверь интернет, правильность конфига и что сервер доступен (порт 51820/UDP).
- **ChatGPT не открывается**: убедись, что используешь профиль «VPN GPT» (тот, что с редиректом через Shadowsocks), а не только «VPN RU».
