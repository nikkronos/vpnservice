## Точечная разведка (лог наблюдений)

- **2026-06-15 — `igareck/vpn-configs-for-russia` (бесплатный агрегатор публичных конфигов, GitHub).** Не сервис-конкурент, а другой формат: подписки-`.txt` (VLESS+Reality / SS / Hysteria2 / Trojan / Tor-bridges) на GitHub raw, авто-обновление каждые 2ч, импорт в Happ/Karing/v2rayN. Бесплатно (Patreon-донат). **Whitelist-обход (Megafon/Yota): CIDR-конфиги через РФ-облачные подсети — VK, Yandex, CDNVIDEO, Beeline** (подтверждает наш `project_vpn_rkn_ip_block`: РФ-облако переживает белые списки). **Слабость = наша ниша:** их же README признаёт — публичные конфиги «быстро появляются и быстро умирают» → нестабильно; серверы неизвестных операторов = риск приватности (honeypot/логирование). Мы = managed / стабильно / per-user / поддержка. **Лид:** CDNVIDEO/Beeline-подсети для гэпа «Мегафон при БС» (ROADMAP ⏸). **Безопасность репо:** только текст/`.md`/QR, без бинарей/скриптов — файл-вируса нет; риск только в доверии чужому exit-серверу (не импортировать в наши клиенты/инфру; тест — на пустом устройстве). Клиент `Happ-proxy` = тот же Happ, что мы рекомендуем.

- **2026-06-02 — «Юбуст» (браузерное расширение, 149₽/мес).** Знакомый владельца пользуется как расширением в браузере. Форм-фактор — расширение, не приложение/`.conf`. Цена 149₽ ниже нашей 200₽, в русле массмаркета (~150-185₽). **Стратегически:** браузерное расширение = *удобство* (одна кнопка, но защищает только трафик браузера, не system-wide), а не *устойчивость* (наша ниша — AmneziaWG/VLESS на всю систему, «работает когда другие отвалились»). Разные лиги: они за «лень настраивать», мы за «у меня ничего не работает». **Связь с идеей «своё клиентское приложение»** (ROADMAP, горизонт 6-12 мес): расширение могло бы быть *более дешёвым первым шагом*, чем полноценный кросс-платформенный клиент. Сейчас far-off — просто фиксируем.

**Реферальная система:** автоссылка → +67₽ на баланс за каждого друга.

**Ключевое преимущество:** цена 100₽/мес + простота. Ключ скопировал — вставил — работает.

### 🥟 VPN Пельмень (не верифицирован, источник — Instagram-реклама)
*Добавлен 2026-05-22. Точное название бренда/бота неизвестно, владелец увидел рекламу в Instagram. Информация ниже — со слов рекламы, требует верификации (попробовать установить, проверить тарифы).*

**Заявленная ключевая фича — "удобный обход":**
> «Заходишь на YouTube — через VPN. Заходишь на Госуслуги — без VPN, не выключая ничего.»

То есть **split tunneling** по доменам: трафик к заблокированным/иностранным сайтам идёт через VPN, к российским — напрямую. Удобство в том, что банковские приложения, Госуслуги, Яндекс Карты и т.п. **продолжают работать с правильной геолокацией** (IP внутри РФ), пока пользователь не выключает VPN.

**Что неизвестно:**
- Цена
- Платформы (вероятно есть своё приложение, поскольку клиентский split tunneling сложно сделать на стандартных VLESS-клиентах)
- Реализация: клиентская (AllowedIPs со списком RU подсетей в исключении) или серверная (router-based, как у нас с `unified_ss_dst`)
- Работа при белых списках РКН / на Мегафон-Yota

**Сравнение с нашим текущим решением:**
- **У нас уже есть инфраструктура для split tunneling** — `unified` профиль (`docs/archive/spec-04-implementation-unified-profile.md`). Серверная реализация через ipset `unified_ss_dst` + iptables REDIRECT 1081 на eu1. Только заблокированные сайты идут через VPN-туннель к ss-redir, всё остальное — напрямую.
- **Не доступно в UI бота** — пользователь получает только обычный `vpn` или `vpn_gpt` профиль. `unified` создаётся вручную владельцем для тестовых случаев.
- **Что нужно сделать:** выдавать `unified` через бот как третью опцию ("📡 Универсальный — VPN для заблокированных + прямой доступ к РФ-сервисам"). Технически — только UI-задача, серверная часть готова.

**Стратегический вывод:**
Эта фича — реально удобная для нетехнического юзера, и она у нас уже технически есть. Не доставать её в UI = недопонимание ценности своего же продукта. Добавлю в ROADMAP P1 после УТП.
