# Инструкция: Как подключить Telegram MTProto Proxy

## Для iOS (iPhone/iPad)

1. Открой приложение **Telegram** на устройстве.
2. Перейди в **Настройки** → **Данные и хранилище** → **Прокси‑серверы**.
3. Нажми **"Добавить прокси"**.
4. Выбери тип **"MTProto"**.
5. Вставь ссылку подключения:
   ```
   tg://proxy?server=185.21.8.91&port=443&secret=29d11c61ea1b644d75299dd0706c2da3
   ```
   Или введи параметры вручную:
   - **Сервер**: `185.21.8.91`
   - **Порт**: `443`
   - **Секрет**: `29d11c61ea1b644d75299dd0706c2da3`
6. Нажми **"Сохранить"**.
7. Включи прокси переключателем.
8. Проверь, что Telegram работает (отправь сообщение, проверь каналы).

## Для Desktop (Windows/macOS/Linux)

1. Открой приложение **Telegram Desktop**.
2. Перейди в **Настройки** → **Продвинутые** → **Сеть и прокси**.
3. Нажми **"Добавить прокси"** или **"Настроить прокси"**.
4. Выбери тип **"MTProto"**.
5. Введи параметры:
   - **Сервер**: `185.21.8.91`
   - **Порт**: `443`

6. Нажми **"Сохранить"** или **"Применить"**.
7. Включи прокси.
8. Проверь, что Telegram работает.

## Альтернативный способ (через ссылку)

Можно просто открыть ссылку в браузере или Telegram:

```
https://t.me/proxy?server=185.21.8.91&port=443&secret=29d11c61ea1b644d75299dd0706c2da3
```

Telegram автоматически предложит добавить прокси.

## Важные замечания

- **Прокси работает независимо от VPN**: можно использовать только прокси для Telegram, не включая VPN.
- **Пинг**: ~45 мс (приемлемо для Telegram).
- **Безопасность**: трафик Telegram шифруется через MTProto‑протокол, сервер не видит содержимое сообщений.
- **Если прокси не работает**: проверь, что параметры введены правильно (особенно секрет — длинная hex‑строка без пробелов).

## Отключение прокси

Чтобы отключить прокси:
1. Открой настройки прокси в Telegram.
2. Выключи переключатель прокси или удали его.

## Вопросы и проблемы

Если возникли проблемы с подключением:
- Проверь, что все параметры введены правильно.
- Убедись, что интернет‑соединение работает.
- Попробуй перезапустить Telegram.
- Если проблема сохраняется — обратись к администратору VPN‑сервиса.
