# Нотация имён профилей VPN

Используется в инструкциях и при общении с пользователями для единообразия.

## Профили WireGuard

| Название в документации | Фактическое имя конфига / туннеля | Назначение |
|------------------------|-----------------------------------|------------|
| **VPN RU** / **Обычный VPN** | `client1`, профили от Telegram‑бота | Трафик через WireGuard напрямую (NAT на сервере), без редиректа через Shadowsocks. Быстрый доступ, YouTube, обычные сайты. |
| **VPN GPT** | `iphone`, `friend-pc`, `friend-iphone`, `friend-ipad` | Трафик через WireGuard; HTTP/HTTPS на сервере редиректится через Shadowsocks. Обход блокировок ChatGPT и др.; поведение YouTube/сайтов зависит от IP Shadowsocks. |

## Другие сервисы

| Название | Описание |
|----------|----------|
| **MTProto‑прокси** / **Telegram‑прокси** | Отдельная ссылка `tg://proxy?...` для Telegram. Работает без включения VPN. |
| **Shadowsocks (на ПК)** | Отдельный клиент на Windows для обхода блокировок поверх «Обычного VPN» (по желанию). |

## Для друзей (примеры)

- **Friend VPN** — обычный VPN (профиль от бота).
- **Friend VPN GPT** — наши профили `friend-pc`, `friend-iphone`, `friend-ipad` с редиректом через Shadowsocks.

Имена файлов конфигов на сервере могут отличаться (`friend-pc.conf`, `iphone.conf` и т.д.); в клиенте пользователь может переименовать туннель для удобства (например, «VPN GPT»).
